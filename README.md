# Software Developer Salary Prediction

## Overview

This project predicts the **annual salary of software developers** using machine learning based on factors such as **country, education level, and years of professional coding experience**.

The application performs **data preprocessing, exploratory data analysis (EDA), model training, and salary prediction** using Python-based data science libraries.

This project demonstrates practical skills in **Python, data processing, machine learning, and building data-driven tools**, which are relevant for data-focused software engineering roles.

---

## Features

* Data cleaning and preprocessing pipeline
* Exploratory Data Analysis (EDA) for understanding salary trends
* Machine learning model for salary prediction
* Interactive interface for user input and predictions
* Visualization of salary distributions and insights

## Installation

Clone the repository

```
git clone https://github.com/PandeySunny/software-developer-salary-prediction.git
cd software-developer-salary-prediction
```

Install dependencies

```
pip install -r requirements.txt
```

---

## Running the Application

```
streamlit run app.py
```

Open in browser:

```
http://localhost:8501
```
